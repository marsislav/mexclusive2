== Description ==
mexclusive2 is a fully responsive and customizable WordPress theme. It is WooCommerce ready, making it suitable for news agencies, personal blogs, magazines, and online shops. All theme settings can be easily customized using the WordPress Customizer (Appearance -> Customize). The theme includes Bootstrap and Font Awesome icons.

== Installation ==
1. Upload the `mexclusive2` folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Themes' menu in WordPress.
3. Customize the theme settings under Appearance -> Customize.
4. Start adding content to your site!
